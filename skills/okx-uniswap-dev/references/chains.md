## Native Token Handling

For native tokens (ETH, MATIC, BNB, AVAX):

- Use `NATIVE` as the `inputCurrency` or `outputCurrency` parameter in swap URLs
- Do NOT use the wrapped token address for native tokens in swap URLs
- Example: Swap ETH -> USDC: `inputCurrency=NATIVE`
- Example: Swap WETH -> USDC: `inputCurrency=0xC02aaA39b223FE8D0A0e5C4F27eAD9083C756Cc2`

## RPC Rate Limits

Public RPCs have rate limits. For production, use premium providers: Alchemy, Infura, QuickNode, or chain-specific premium RPCs.
